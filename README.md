# zerobaker

这是王珺怡的个人网站，也是“一家只在网上营业的烘焙坊”。网站用于展示经历、作品、兴趣、审美与 Vibe Coding 探索。

## 当前阶段

项目目前完成了基础初始化：

- 单页网站的五个区域：主页、自我介绍、作品展示、烘焙秘方、联系方式；
- 透明导航栏和页面内平滑跳转；
- 适配电脑和手机的基础排版；
- 作品、秘方和联系方式的数据结构；
- 清楚命名的图片占位路径。

当前版本只是网站骨架，不是最终视觉稿。

## 文件说明

- `index.html`：网页内容的骨架。
- `style.css`：颜色、排版、基础动画和手机端适配。
- `script.js`：导航与内容生成等基础交互。
- `content.js`：以后最常修改的内容数据。
- `assets/`：图片、音频和公开文件。
- `docs/PROJECT_BRIEF.md`：唯一的正式项目说明文件。
- `docs/`：风格指南、内容规划、任务进度和其他项目文档。

## 在电脑上预览

请在 VS Code 中打开这个项目文件夹，再打开“终端”。确认终端当前位于项目根目录后，运行：

```bash
python -m http.server 8000
```

如果电脑上的 Python 命令是 `python3`，则运行：

```bash
python3 -m http.server 8000
```

然后在浏览器访问：

```text
http://localhost:8000
```

停止本地预览时，回到终端按 `Control + C`。

## 修改内容

作品、烘焙秘方、联系方式和部分基础文字集中在 `content.js`。后续补充真实资料时，优先修改该文件，不需要直接复制大量 HTML 卡片。
