# STYLE_GUIDE.md

## 当前用途

这份文件记录网站的长期视觉原则。当前初始化版本只建立基础变量和排版，不代表最终视觉已经完成。

## 品牌书写与命名

- 对外展示的品牌名统一写作 `ZeroBaker`，不得写成 `zerobaker`、`Zerobaker` 或其他大小写形式；
- 文件名和代码变量继续使用小写英文；多个单词的文件名使用短横线连接，例如 `zero-baker-logo.png`；
- 品牌展示规范与代码命名规范分开处理，不因页面上的正式写法而修改既有代码命名方式。

## 视觉关键词

- 温暖；
- 手作感；
- 纸张拼贴；
- 有趣但不幼稚；
- 个性鲜明但不影响阅读；
- 像个人数字空间，不像通用程序员模板或粉丝主页。

## 基础配色

- 奶油白：页面主背景；
- 草莓红：主要强调色与重要链接；
- 抹茶绿：次级装饰；
- 蓝莓紫：次级装饰；
- 焦糖棕：文字与烘焙主题细节；
- 深棕色：正文主色。

具体色值集中在 `style.css` 顶部的 `:root` 变量中，后续可以统一调整。

## 字体原则

- 正文优先使用系统自带中文字体，保证清楚易读和加载稳定；
- 标题与装饰文字可以在后续加入手写风格字体；
- 手写字体不能用于大段正文；
- 在没有确认字体授权和加载方式前，不下载或引入外部字体。

## 双语标题原则

- 四个主要板块统一使用中英文双标题：
  - 自我介绍 / ABOUT THE BAKER
  - 作品展示 / FRESHLY BAKED
  - 烘焙秘方 / SECRET RECIPES
  - 联系方式 / FIND ME
- 英文副标题统一使用以上大写形式；
- 中英文标题应有清楚的主次层级，同时保持手机端可读，不为装饰而压缩字号或间距。

## 图片原则

- 图片使用相对路径；
- 图片保持原比例，不拉伸；
- 公开前检查照片隐私和内容授权；
- 第一阶段只使用清楚命名的占位路径；
- 文件名使用小写英文和短横线，例如 `jeju-back-view.jpg`。
- 首页四块蛋糕图片后续单独生成，在正式素材完成前继续保留当前占位路径，不用临时图片替换。

## 排版原则

- 页面为连续滚动的单页结构；
- 每个板块使用 `min-height`，避免小屏幕内容被截断；
- 电脑端和手机端同步考虑；
- 装饰元素不得遮挡正文和按钮；
- 内容区保持适当留白，让访客能快速理解层级。

## LAYOVER 表达原则

- 首页首先表达“短暂停留、回望、反思、疗愈与生活注脚”的含义；
- 金泰亨首张个人专辑是其灵感来源，可在“烘焙秘方”或个人故事中进一步说明；
- 首页不应让专辑信息先于站主本人的生活叙事，也不应呈现为粉丝主页。

## 动效原则

- 当前阶段仅使用平滑滚动与轻微悬停反馈；
- 复杂纸团互动后续单独开发；
- 手机端必须提供不依赖鼠标的操作方式；
- 尊重系统的“减少动态效果”设置。
